##  Album
Album related permissions:
	- `album.manage` : create, modify, delete any album.
	- `album.see-hidden` : can see hidden album.
##  Track
Track related permissions:
	- `track.manage` : create, modify, delete any track.
##  Review
Review related permissions:
	- `review.manage` : create, modify, delete any Review.
##  Group
Group related permissions:
	- `group.manage` : create, modify, delete any group.
##  User
User related permissions:
	- `user.manage` : create, modify, delete any user.
